## Requisitos

- Python 3.9 ou superior
- PostgreSQL

## Como Usar

1. Clone este repositório:
    ```bash
    git clone https://github.com/JoaoPPagotto/exame-backend-dtlabs-2025.
    ```

2. Instale o Python 3.9 ou superior, caso ainda não tenha.
   

3. Execute o projeto:
    ```bash
    docker-compose up --build
    ```
